# OpenALP v0.1.1 Release Notes

OpenALP v0.1.1 updates the MVP with clarified installation/discovery and finalized licensing alignment.

## Included updates in v0.1.1

1. Apache-2.0 license finalized.
2. README licensing text corrected.
3. Installation guide updated for `openalp-v0.1.1.tar.gz`.
4. Agent-first discovery assets added:
- `agent-manifest.json` (machine-readable install/execution contract)
- `web/agent.html` (agent bootstrap page)

## Core capabilities retained

1. `.alp` executable format
2. `.alpb` raw source format
3. deterministic runtime + validation
4. checksum/signature + handshake + determinism tooling
5. browser demos and interop docs
