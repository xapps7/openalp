# Changelog

## v0.1.0-mvp

### Added
- OpenALP binary module format (`.alp`)
- Raw machine-style source format (`.alpb`)
- Deterministic stack VM runtime
- CLI commands for validate/run/inspect/decompile/build-raw
- Integrity checks (`checksum`)
- HMAC signature sidecars (`sign`)
- Demo public-key signature flow (`keygen-demo-rsa`, `sign-rsa`)
- Determinism reports
- Browser runner and landing page
- Onboarding, agent context, interop docs, and launch tooling
